
# Car Dealership Business App

A modern small business car dealership management frontend project built with React.

## Features
- Car inventory listing
- Featured vehicles section
- Car cards with pricing and specs
- Search-ready structure
- Responsive UI
- GitHub Actions CI workflow
- Clean folder architecture

---

## Tech Stack
- React
- JavaScript
- CSS
- GitHub Actions

---

## Preview Features
- Inventory dashboard
- Featured cars
- Business branding
- Reusable components
